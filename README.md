## Jekyll

Jekyll 要求博客文章的文件名按照以下格式命名：

`YEAR-MONTH-DAY-title.MARKUP`

* **YEAR** 4位数字，表示年份
* **MONTH** 2位数字，表示月份
* **DAY** 2为数字，表示日期
* **MARKUP** 表示文件扩展名

Jekyll 还提供了对代码片段的强大支持：


```ruby
def print_hi(name)
  puts "Hi, #{name}"
end
print_hi('Tom')
#=> prints 'Hi, Tom' to STDOUT.
```
